# Análisis de negocio para una empresa de entretenimiento - TripleTen - Bootcamp Análisis de Datos

El objetivo del proyecto era elaborar indicadores claves de desempeño (KPI) para tomar decisiones basadas en datos que ayuden a mejorar el rendimiento de la empresa. 

Para ello dividí a los clientes en cohortes para identificar las preferencias de cada una de ellas, asimismo, realicé cálculos para identificar la tasa de retención y cancelación.

Con base en los hallazgos identificados, pude identificar cuales estrategias de marketing son más rentables apra la compañía y cuales no generaban ingresos.


Herramientas utilizadas: Python
