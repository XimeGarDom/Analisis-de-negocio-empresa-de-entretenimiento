# Análisis de negocio para una empresa de entretenimiento

El proyecto tiene el objetivo de brindar una estrategia de negocios a la empresa Showz para maximizar sus ingresos, verificando las estretegias de marketing de la empresa. 
Se llevaron a cabo las siguientes actividades: 
- Analizar cohortes.
- Identificar la tasa de retención y cancelación

Herramientas utilizadas: Python
